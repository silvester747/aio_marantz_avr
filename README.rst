AsyncIO access to Marantz AVRs.


* Free software: MIT license
* Documentation: https://aio-marantz-avr.readthedocs.io.


Features
--------

* Control and read status of a Marantz AVR over Telnet using python AsyncIO.
* Command line tool.

Supported models:

* SR7011 (tested)
* AV7703
* SR6011
* SR5011
* NR1607
